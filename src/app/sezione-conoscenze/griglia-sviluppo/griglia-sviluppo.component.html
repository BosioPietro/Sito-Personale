Applicazioni Desktop

Sviluppo di applicazioni desktop con C# sia di tipo Console che con interfaccia grafica. Applicazioni GUI create con Windows Form o con UWP per una maggiore compatibilità.

Sviluppo SQL

Sviluppo e gestione di database SQL utilizzando DBMS tra cui Microsoft SQL Server. Implementazione di funzionalità avanzate come trigger e stored procedures.

Applicazioni Mobile

Creazione di applicazione mobile, per Android e iOS, con un'unico codice sorgente usando piattaforme di sviluppo come Capacitor (in combinazione con Ionic) e Cordova.

Progettazione Database

Progettazione e sviluppo di database utilizzando schemi ER e schemi logici. Creazione e documentazione di strutture e relazioni dei dati tramite diagrammi UML per una chiara rappresentazione e gestione delle informazioni.
